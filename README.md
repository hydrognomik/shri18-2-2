## Запуск

```
npm i

npm run dev
```

## Сборка

```
npm run duild
```

### Описание

* Использовал SCSS из-за циклов, переменных и `&` синтаксиса.
* Сборка с помощью `gulp`.
* Мобильные стили отделены от десктопных.
* Крутилку тёплого пола нарисовал на svg.
